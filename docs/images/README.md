# Project artwork

- `hero.svg` / `hero.png`: README banner and optional GitHub social-preview artwork.
- `architecture.svg` / `architecture.png`: shared TCP/RTU software architecture diagram.
- `request-flow.svg` / `request-flow.png`: Modbus TCP ADU processing path.
- `rtu-timing.svg` / `rtu-timing.png`: T1.5/T3.5 receive timing and invalid-gap behavior.
- `register-map.svg` / `register-map.png`: default data-bank illustration.

The SVG files are the source artwork. PNG exports are included for platforms that do not render SVG.
